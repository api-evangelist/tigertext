# TigerTextThis is a repo for managing machine readable artifacts for the [TigerText APIs](http://www.tigertext.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [TigerText APIs](http://www.tigertext.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.tigertext.com/)- [Developer](http://www.tigertext.com/)- [Twitter](https://twitter.com/tigertext)- [GitHub](https://github.com/tigertext)The [APIs.json](https://github.com/api-evangelist/tigertext/blob/master/apis.json) index provides links to all the relevant links for the TigerText APIs, including the OpenAPI and Postman if they are present.
